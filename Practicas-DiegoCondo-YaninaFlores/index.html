HTML5

Se trata de una nueva versión de HTML, con nuevos elementos, atributos y comportamientos. Contiene un conjunto más amplio de tecnologías que permite a los sitios Web y a las aplicaciones ser más diversas y de gran alcance. A este conjunto se le llama HTML5.

CSS3

CSS son las siglas de Cascading Style Sheets -Hojas de Estilo en Cascada- que es un lenguaje que describe la presentación de los documentos estructurados en hojas de estilo para diferentes métodos de interpretación, es decir, describe como se va a mostrar un documento en pantalla, por impresora, por voz (cuando la información es pronunciada a través de un dispositivo de lectura) o en dispositivos táctiles basados en Braille.
Al hacer click en el logo usted puede dirigirse a la página de masadelante.com.En donde podrá conocer las Ventajas de CSS3

JavaScript

Al igual que HTML, Javascript es un lenguaje de programación que se puede utilizar para construir sitios Web y para hacerlos más interactivos. Comparte muchas de las características y de las estructuras del lenguaje Java, sin embargo fue desarrollado independientemente. El lenguaje Javascript puede interactuar con el código HTML, permitiendo a los programadores web utilizar contenido dinámico. Por ejemplo, hace fácil responder a los acontecimientos iniciados por usuarios (como introducción de datos en formularios) sin tener que utilizar CGI.
Haciendo click en el logo podrá ingresar a masadelante.com. Encontrará contenido de ¿Qué es JavaScrip? - Definición de JavaScript.
